# 四码仓颉（Sicang5）巅峰重构计划 (v6)

这一版是真正的质变！我们不再死守“首码”这一个维度，而是参考了主流形码（如五笔、郑码）的“阶梯式权重”逻辑，将中文最核心的高频字均匀地分布在了 25 个按键上。

## 1. 一简规则的“阶梯竞争”机制
我们将按键的分配分为三个优先级，每个级别都有特定的“保护门槛”：
- **T1 级：原字根 (如：木)**。拥有基准权重。
- **T2 级：首码匹配 (如：本 dm)**。如果频次超过 T1 级的 **1.5 倍**，则成功篡位。
- **T3 级：包含匹配 (如：的 hqi 包含 a)**。
  - 这是专门为“全明星字（Top 50）”准备的通道。
  - 只要字码中**包含**该字母，且频次是当前 T2 级最优解的 **3 倍**，即可“跨级抢位”。
  - 这种逻辑确保了像“的”、“不”、“在”、“就”这些出现率极高的字，即便不是首码，也能根据其形状特性找到最舒服的按键。

## 2. 预期的“全明星”布局
- `a` (日) -> **的** (rank 1)：虽然是 `hqi`，但它包含 `a` 且频次无敌。
- `f` (火) -> **不** (rank 6)：`不 (mf)` 包含 `f`。
- `g` (土) -> **在** (rank 7)：`在 (klg)` 包含 `g`。
- `u` (山) -> **就** (rank 11)：`就 (iuy)` 包含 `u`。
- `r` (口) -> **和** (rank 12)：`和 (hdr)` 包含 `r`。

## 3. 细节与清洗
- **Header 统一**：`sicang5_z.txt` 标题改为 `# 原字根`；`sicang5_3.txt` 标题改为 `# 三简`。
- **代码注释**：`scripts/gen_sicang5.py` 全中文注释，包含这一套阶梯竞争算法的详细实现。
- **二三简策略**：维持 Top 1000 精英策略，保证码表清爽、冲突极低。
