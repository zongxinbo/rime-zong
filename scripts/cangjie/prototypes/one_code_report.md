## 策略

- 权重：知乎 33%，北语 27%，台标 22%，古籍 18%。
- 默认先出普通位，`x/z` 仅在追加模式下放到报告末尾。

- 字根字/首码匹配优先；高频包含键位可跨位；普通简码难覆盖的字获得加权。
- 简码可得性：只看 `cangjie5.dict.yaml` 源码表；二码/三码前缀若被高频原生字占据，则提高候选优先级。
- 当前正式版来自 `one_code.txt`，最终是否采用建议仍需人工定稿。
