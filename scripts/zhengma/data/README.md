    （1）本拆分表包含汉字基本集CJK、扩展集CJK-A~J全汉字，所有涉及字型以大陆宋体为准，部分超集汉字同时兼顾中易宋体、书局方正、Unicode官方字样等字型。
    （2）拆分表将各汉字拆分成字根，所有字根均包含在“字根表.txt”文件内。其中，非成字的字根以“{□□}”的组合形式表示，手机无法显示的大字集字根采用形近字根表示。
    （3）递进拆分仅将汉字拆分至汉字部件。
    （4）“㳤厯”等汉字，因取根有分歧，作兼容拆分。
    （5）“㥁値塡巓悳愼槇眞鎭顚鷆”等汉字，因根序有分歧，作兼容拆分。
    （6）“乄”字，因“㇢”笔画在处理上有分歧，作兼容拆分。
    （7）“夨奊捑”等汉字，按国标笔顺取根，不再作兼容拆分。
    （8）为方便输入西夏文，特增设八个西夏文专用字根（单独成字时补aa或aaa）。
