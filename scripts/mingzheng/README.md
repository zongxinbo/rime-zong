# 明郑

明郑是基于郑码资料重整的新形码方案，目标是在保留郑码取形基础的前提下，重新梳理简繁字形、简码和词码规则。

## 目标

- 简繁字根分流，避免简繁字形因字根归并产生不必要重码。
- 简码与全码保持可学习的一致性，优先让简码成为全码前缀。
- 字码与词码保持一致，降低造词码和单字码之间的割裂。
- 保留郑码既有资料中可复用的拆分、字根和单字编码信息。

## 名称

- 中文名：明郑
- 英文名：Ming Zheng
- schema id：`mingzheng`

## 数据来源

初始资料来自 `scripts/zhengma/data/`：

- `chars.txt`：单字码表
- `split.txt`：拆分表

明郑会单独维护字根表：

- `data/roots.txt`：明郑字根表，基于郑码字根表调整

通用 IDS 资料保留在 `scripts/ids/ids.txt`。
